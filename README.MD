## Simple shapes library

### Requires
- net7.0

### Installation
1. Clone the repository
   ``` shell
   git clone https://github.com/AntonYartsev/shaper.git
   ```
2. Restore dependencies
   ``` shell
   cd shaper
   dotnet restore
   ```
3. Build the project
   ``` shell
    dotnet build
    ```

### Run tests
   ``` shell
   dotnet test
   ```

### Run play app
   ``` shell
    dotnet run --project play
   ```
